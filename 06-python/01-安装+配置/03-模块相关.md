# 03-模块相关

## 命令

| 命令                            | 说明                                                        |
| ------------------------------- | ----------------------------------------------------------- |
| pip install --upgrade 模块名    | 升级模块                                                    |
| pip uninstall 模块名            | 卸载模块                                                    |
| pip freeze                      | 列出当前环境中所有包及其版本（常用于生成 requirements.txt） |
| pip freeze > requirements.txt   | 生成 requirements.txt                                       |
| pip install -r requirements.txt | 根据文件批量安装依赖                                        |
